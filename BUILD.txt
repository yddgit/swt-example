
 Building
 --------
 
 To build this project you will need Maven 3. You can get it at:
 
     http://maven.apache.org

 Clean compilation products:
 
     mvn clean
     
 Compile:
 
     mvn compile
     
 Run from the java method:

     com.my.explorer.Explorer.main()
     com.my.hello.HelloSWT.main()
